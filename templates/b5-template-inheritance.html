Step 5 - Template inheritance

Create connections between the different pages

  1. In app.py, create the routes for the other pages
  2. In app.py, from flask import url_for. This will connect our html links to the routes in order to create links between 2 different pages.
  3. Inside html files, links to another page should have the href ='{{ url_for('add_pet') }}'. be careful, it is the NAME OF THE FUNCTION, not the name of the file or of the route !

DRY up your HTML code

The head section in our html files is repeated in every file. Flask gives us the ability to place these repeated items into their own file and then use them whenever we need to = template inheritance.
