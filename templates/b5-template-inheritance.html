Create connections between the different pages

  1. In app.py, create the routes for the other pages
  2. In app.py, from flask import url_for. This will connect our html links to the routes in order to create links between 2 different pages.
  3. Inside html files, links to another page should have the href ='{{ url_for('add_pet') }}'. be careful, it is the NAME OF THE FUNCTION, not the name of the file or the route !

DRY up your HTML code

The head section in our html files is repeated in every file. Flask gives us the ability to place these repeated items into their own file and then use them whenever we need to = template inheritance.

  1. Create a layout.html file
  2. Layout.html contains all that is the same in each html file. And add block to indicate where the content, the buttons... will be displayed.
    • {% block button %}{% endblock %}
    • {% block content %}{% endblock %}
    • {% block link %}{% endblock %}
  3. Change your html files with button block, content block...
    • {% extends 'layout.html' %}
    • {% block button %}{% endblock %}
    • {% block content %}{% endblock %}
    • {% block link %}{% endblock %}

This creates the structure of our page while also keeping our html DRY. Cool !
